# TaskLedger Legal Pages

Static pages for TaskLedger legal documents, ready to publish with GitHub Pages.

## Files
- `index.html` - landing page with links to the legal documents
- `privacy-policy.html` - Privacy Policy for TaskLedger
- `terms-of-use.html` - Terms of Use for TaskLedger
- `styles.css` - shared styling for the site

## Before publishing
Replace these placeholders in both legal pages:
- `REPLACE_WITH_SUPPORT_EMAIL`
- `REPLACE_WITH_GITHUB_PAGES_URL`

## GitHub Pages
In the repository settings, enable **Pages** and publish from the default branch root. After that, your URLs will look like:
- `https://YOUR_USERNAME.github.io/TaskLedgerPrivacyPolicy/privacy-policy.html`
- `https://YOUR_USERNAME.github.io/TaskLedgerPrivacyPolicy/terms-of-use.html`
